# Делегирование и план

Цель: быстро разбить задачу на выполнимые куски и раздать их доступным агентам.

1. Выдели независимые подзадачи.
2. Назначь каждую подзадачу наиболее подходящему агенту.
3. Следи за активными run и зависимостями.
4. Не зови саб-агентов для того, что можно закрыть одним коротким шагом.
