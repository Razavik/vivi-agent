# Навык: работа с адресной строкой браузера

Цель: открыть URL или поисковый запрос в браузере устойчиво.

Предпочтительный путь, если доступен `open_url`:
1. Используй `open_url(url)`.
2. Проверь загрузку через screenshot или UI Automation.

Если нужно работать именно с текущим окном:
1. Нажми `system_key_press("ctrl+l")`.
2. Введи URL/запрос через `system_type_text`.
3. Нажми `system_key_press("enter")`.
4. Проверь загрузку новым наблюдением.

Не кликай по адресной строке мышью, если можно использовать `ctrl+l`.
