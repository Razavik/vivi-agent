# Навык: клик по кнопке или пункту меню

Цель: нажать кнопку/пункт меню структурно, а не по пикселям.

1. Вызови `list_ui_elements(query?, max_results?)`.
2. Ищи `Button`, `MenuItem`, `TabItem`, `Hyperlink`, `ListItem` или элемент с подходящим `name`.
3. Если есть уверенный кандидат, вызови `click_ui_element(id)`.
4. Проверь результат: экран изменился, меню открылось/закрылось, активная вкладка сменилась, появилась новая область.

Если есть несколько похожих кандидатов:
- выбери тот, чей `rect` ближе к ожидаемой области;
- при сомнении сделай screenshot/crop и сопоставь с визуальным положением.
