# Навык: проверка результата

После действия выбери проверку по типу задачи:

- Ввод текста: виден введённый текст, каретка, активное поле или новое значение в UI.
- Клик по кнопке: изменилось состояние, открылось меню/страница/диалог, исчезла кнопка загрузки.
- Навигация: изменился заголовок окна, URL, контент страницы или активная вкладка.
- Файловая/системная операция: есть код возврата, файл, процесс, порт или отдельное наблюдение.

Если проверка неоднозначна, сделай ещё одно наблюдение вместо финального ответа.
