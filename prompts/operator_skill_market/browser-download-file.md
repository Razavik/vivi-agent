# Скачивание файла в браузере

Цель: довести скачивание до проверяемого результата.

1. Запусти действие скачивания.
2. Проверь визуально, что браузер начал загрузку или показал подтверждение.
3. Если появился системный диалог, используй UI Automation или клавиатуру для подтверждения.
4. После завершения проверь папку загрузок через файловый инструмент, если он доступен.
5. Сообщи путь к файлу, а не только факт скачивания.
