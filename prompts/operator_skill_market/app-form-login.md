# Авторизация в приложении

Цель: аккуратно заполнить форму входа без лишних кликов.

1. Найди поля через `list_ui_elements`: email/login, password, кнопка входа.
2. Сфокусируй первое поле через `focus_ui_element(id)`, введи логин.
3. Перейди в пароль через UI Automation или `tab`, введи пароль.
4. Нажми кнопку входа через `click_ui_element(id)`.
5. Проверь результат: исчезла форма, появился профиль/рабочая область или сообщение об ошибке.

Не вводи пароль повторно, если не проверил, что поле действительно активно.
