# Заполнение веб-формы

Цель: заполнить несколько полей формы с проверкой каждого шага.

1. Получи список элементов формы через `list_ui_elements`.
2. Для каждого поля: сфокусируй, введи значение, проверь, что значение появилось.
3. Для выпадающих списков предпочитай клавиатуру после фокуса.
4. Перед отправкой проверь обязательные поля и видимые ошибки.
5. Отправь форму через найденную кнопку, затем проверь результат.
