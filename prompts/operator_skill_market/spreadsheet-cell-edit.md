# Редактирование ячейки таблицы

Цель: изменить значение в таблице без промаха мышью.

1. Сначала добейся фокуса нужной таблицы.
2. Для переходов используй стрелки, tab, enter или поиск, если приложение его поддерживает.
3. Чтобы заменить значение, нажми `F2` или начни ввод в активной ячейке.
4. Введи значение через `system_type_text`.
5. Подтверди Enter и проверь видимое значение.
